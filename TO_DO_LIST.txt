This lists specifies the design steps neccessary to create a more interactive PetroWebApp (working title)

GOALS:
(repuired)
1) The WTI crude oil index ticker should be clickable and dragable 
2) Circle radii/colors should respond to the movement of the ticker accordingly
3) Bar chart should respond to the movement of ticker accordingly
4) Bar chart elements should show raw data value when moused over
(optional)
5) Bar chart elements should re-sort themselves on each transition so so that the largest value is on top
